CREATE EVENT


Event Name is required.
Event Name atleast 5 characters.
Event Name must be at most 20 characters long.
Event Name should contains characters only.

Skill Set is required.
Skill Set atleast 10 characters.
Skill Set must be at most 40 characters long.
Skill Set should contains characters only.

Count Of Panel is required.
Count Of Panel must be single digit and not zero (0)
Count Of Panel should single digit only.
Event Date is required.
Format Date dd/MM/yyyy allowed.
Escalation Date is required.
Format Date dd/MM/yyyy allowed.
